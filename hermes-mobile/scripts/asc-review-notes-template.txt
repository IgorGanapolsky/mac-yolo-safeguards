This build includes a built-in demo mode for review — no account, gateway, or setup needed.
1. Install the build, then open hermes://setup?demo=1 in Safari (or tap from Notes).
2. Return to Hermes Mobile — demo chat, approval prompts, and Leash UI preview are enabled.
3. In Chat, send a message and tap Approve or Deny on any prompt.
4. Open Leash to preview the subscription gate rules UI.
Alternatively: Settings → turn ON "Demo mode."
In normal use, a user connects the app to their OWN computer (macOS, Linux, or Windows) by
scanning a QR code from their Hermes gateway. The demo simulates that, so no external server
is involved.
