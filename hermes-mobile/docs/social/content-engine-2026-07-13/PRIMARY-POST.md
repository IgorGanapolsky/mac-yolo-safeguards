## Title

Show HN: Hermes Mobile – phone approve/deny for coding agents on your Mac

## Body

I run coding agents on a Mac and kept getting burned by irreversible tool calls while away from the desk — force-push, destructive shell, deploy scripts.

**Hermes Mobile** is a phone operator UI for agents already running on **your** computer (macOS / Linux / Windows behind a Hermes gateway):

- Chat to your gateway over home Wi‑Fi or Tailscale (off-LAN needs Tailscale/tunnel — not “universal connectivity”)
- Leash approve/deny cards for gated tool calls — **10 routed approvals/week free** (enforced in-app), then Leash Pro **$19.99/mo** for unlimited mobile approvals + standing rules
- QR pair; API keys stay with your gateway — this is not another cloud-agent inbox

Honest limits: Android is on Google Play today; iOS is **not** App Store–live yet (iTunes lookup resultCount=0). Continuous device E2E is currently failing on my side, so I’m not claiming “works on device verified.” Early days; ~no public review base — I’d rather hear where this falls short than collect polite upvotes.
