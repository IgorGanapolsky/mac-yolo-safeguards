## Description (short)

Hermes Mobile is the phone operator console for coding agents running on **your** machine. Chat for free. Leash adds mobile approve/deny (10 routed approvals/week free, then $19.99/mo Pro). Pair with QR; use Tailscale off home Wi‑Fi. Not another cloud agent inbox.
