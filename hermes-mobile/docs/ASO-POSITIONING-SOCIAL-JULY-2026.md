## Executive summary

| Reality | Implication |
|---------|-------------|
| "hermes mobile" Play search returns **12+ unrelated apps** | Cannot own head term overnight — **brand + wedge keywords** in title/short desc |
| **Hermes Agent** (`com.hermesagent.android`) has **1,212 reviews, 4.5★** | Wins on social proof; different product (AI **on** Android) |
| **Hermes-Relay** (Axiom-Labs) = cloud relay "agent in pocket" | Our wedge: **YOUR Mac/Linux/Windows**, YOUR keys, **no cloud credits** |
| Play 2026 weights **D30 retention, install velocity, conversion** | Copy enters candidate pool; **distribution + retention** win ranking |
| Gemini Discovery favors **intent-matching prose** | Target long-tail ("approve Cursor from phone") not bare "hermes" |

**Honest goal:** Climb "hermes mobile" over **weeks**, not days. Near-term: long-tail discovery, branded defense, social backlinks, conversion lift.

## 1. Play Store ASO ranking factors (July 2026 — 12 sources)

### Metadata

| Field | Limit | Weight | Source |
|-------|-------|--------|--------|
| App title | 30 | Highest | [ASOtext](https://asotxt.com/wiki/google-play-search-algorithm), [Sonar](https://trysonar.app/blog/android-aso-guide) |
| Short description | 80 | Second-highest | [IOn Project 2026](https://theionproject.com/blog/google-play-search-algorithm-2026/) |
| Full description | 4000 | Indexed; 2–3% density max | [vmobify 2026](https://vmobify.com/blog/google-play-algorithm-2026/) |
| Developer name | 64 | Indexed | Sonar |

### Post-install signals

Install velocity, D1/D7/D30 retention, conversion rate, review keywords, Android Vitals, uninstall rate — [ASOtext](https://asotxt.com/wiki/google-play-search-algorithm), [ASO World I/O 2026](https://asoworld.com/en/blog/google-i-o-2026-key-developer-insights-for-apps-and-games-on-google-play/).

### External / social

Play: moderate backlink effect. iOS: minimal direct. Social mention velocity + click-through correlates with relevance — [SEO Engico 2026](https://seoengico.com/blog/app-store-optimization-seo-pros-2026), [Authority Specialist](https://authorityspecialist.com/learn/tutorial/how-to-optimize-app-store-seo).

### Branded vs generic (name collision)

- Own brand in title; **don't repeat `hermes` in iOS keyword field** — [Sajid Hasan](https://blog.sajidhasan.com/aso-blueprint-brand-vs-generic-keywords)
- **Never competitor trademarks** in metadata — [Gummicube](https://www.gummicube.com/blog/targeting-competitor-ios-app-brands-with-keyword-optimization)
- Target long-tail intent, not generic "hermes" — [RespectASO](https://respectaso.com/glossary/brand-keywords/)

### Cross-posting

SEO compounds; LinkedIn native long posts; Reddit value-first; Dev.to/Hashnode technical how-tos — [ASOhack](https://asohack.com/blog/aso-final-thoughts-content-distribution).
