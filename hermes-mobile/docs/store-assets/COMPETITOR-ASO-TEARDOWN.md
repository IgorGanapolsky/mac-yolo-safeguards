# Competitor ASO teardown — Hermes Mobile

Research date: 2026-07-06. See [STORE-ASO-JULY-2026.md](../STORE-ASO-JULY-2026.md) §5 for summary matrix.

## Termius

- **Play:** [com.server.auditor.ssh.client](https://play.google.com/store/apps/details?id=com.server.auditor.ssh.client)
- **Rating:** 4.6★, 1M+ downloads, 42K reviews
- **Title pattern:** "[Brand] - [Category]" — functional SEO
- **Screenshot angle:** Terminal sessions, themes, multi-device sync
- **Monetization:** Subscription Pro — frequent review complaints about paywall
- **Hermes counter:** "Not SSH — agent approval gate" in screenshot 2 caption

## Moshi

- **Site:** [getmoshi.app](https://getmoshi.app/)
- **Rating:** 4.8★ App Store (750+ ratings cited on site)
- **Title pattern:** Emotional — "baby monitor for AI agents"
- **Screenshot/video:** Terminal + inbox + lifestyle (couch, beach)
- **Free tier:** Full SSH, 2 connections — builds trust
- **Pro:** Mosh, unlimited connections, diff viewer, agent hooks
- **Hermes counter:** Native gateway pairing (no SSH keys), Leash approvals UI, ThumbGate memory

## OpenAI Codex (ChatGPT mobile)

- **Distribution:** No standalone listing — inside ChatGPT app
- **Promise:** QR pair per host, threads, approvals, live diffs from phone
- **Gap Hermes fills:** Multi-agent, own Mac gateway, ThumbGate, gate rules, non-OpenAI stack

## meshTerm

- **App Store:** SSH/FTP via Tailscale
- **Angle:** Network-first (Tailscale magicDNS)
- **Hermes counter:** Leash + chat + honest cellular copy (screenshot 6)

## Replit mobile

- **Play category:** Developer tools, "Vibe Code with AI"
- **Angle:** Cloud IDE on phone
- **Hermes counter:** Local Mac compute, credentials stay on your machine

## ASO tactics to copy

| Tactic | Who does it well | Hermes action |
|--------|------------------|---------------|
| Outcome-first screenshot 1 | Moshi | Ship captioned hero frame |
| Honest free tier in short desc | Moshi | Already in full desc — move to short desc A/B |
| Social proof | Termius (1M+) | Add "594 tests pass" / E2E badge after launch week |
| Agent-specific keywords | Moshi blog SEO | Variant B title/subtitle test |
| Video autoplay hook | Codex/Apple demos | 22s approval-first preview |

## ASO tactics to avoid

| Anti-pattern | Example | Why |
|--------------|---------|-----|
| Generic "SSH client" positioning | Termius | Commodity; Hermes is agent safety |
| Subscription backlash | Termius reviews | Emphasize free approvals forever |
| Feature list short description | Termius Play | Use outcome lines ≤80 chars |
| Logo-only video open | Many indies | Apple rejects low-conversion previews |
