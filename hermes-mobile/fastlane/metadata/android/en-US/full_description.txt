Control an AI coding agent from your phone on your own Mac, Linux, or Windows computer. Chat with the Hermes gateway, see honest run state, and use Leash Pro to approve or deny risky commands before they execute.

Hermes Mobile is an independent mobile control plane for a Hermes gateway you operate. It is built for developers whose workflows also involve Cursor, Claude Code, ChatGPT or OpenAI Codex, Gemini CLI, Replit Agent, GitHub Copilot, Windsurf, OpenClaw, and other coding agents.

Hermes Mobile does not sign in to or replace those products. It solves a different job: supervising the Hermes agent and tools running through your own gateway from an Android phone.

HOW THIS DIFFERS FROM OTHER "HERMES" APPS

• Termux (Android CLI): Runs Hermes on your phone. Hermes Mobile does not — your gateway stays on your computer for better battery, persistence, and full tools.

• Hermes Agent (Hen Works): AI on your Android phone. Hermes Mobile is the opposite — remote control for desktop agents on Mac, Linux, or Windows.

• Hermes-Relay (cloud relay): Routes through a cloud "agent in pocket." Hermes Mobile pairs directly to your gateway via QR, LAN, or Tailscale — no cloud credits.

• Hermex / HermesPilot (native clients): Other Method A apps (WebUI URL + password or Hermes Link relay). Hermes Mobile connects to your gateway API with QR pair, Leash Pro approvals, and honest connection state.

FAQ

• Do I need Termux? No. Hermes runs on your Mac, Linux, or Windows computer; this app is the remote control.

• Is this Hermes Agent (the 1,200★ Play app)? No — that runs AI on Android. We control desktop agents.

• Hermex or HermesPilot? Those are other native clients (WebUI server vs Hermes Link relay). We pair directly to your gateway plus ThumbGate Leash.

• Do I need a cloud account? No Hermes Mobile account required. Optional Leash Pro subscription for gate rules.

• iPhone? Yes — live on the App Store as “Hermes Mobile: AI Agent Leash”. Android is live on Google Play. Both support Leash Pro $19.99/mo.
