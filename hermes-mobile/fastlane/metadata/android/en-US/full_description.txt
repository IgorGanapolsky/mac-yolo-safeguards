Stop runaway AI coding agents from freezing your Mac. Hermes Mobile is a phone remote for agents on YOUR computer — not another cloud-credit chat app. Approve or deny risky tool calls from the couch while keys stay on your machine.

Hermes Mobile is an independent mobile control plane for a Hermes gateway you run on Mac, Linux, or Windows. Built for developers who also use Cursor, Claude Code, ChatGPT / OpenAI Codex, Gemini CLI, GitHub Copilot, Windsurf, Replit Agent, OpenClaw, and similar coding agents.

Hermes Mobile does not replace those products. It solves a different job: supervise the Hermes agent and tools on your own gateway from Android — with honest connection state (no fake green “Connected”).

WHY THIS EXISTS
• Local agents burn tokens, spin fans, and freeze boxes while you are away from the desk
• Cloud agent apps move your work off your machine; Hermes Mobile keeps the runtime on your computer
• You need approve/deny for dangerous tools without SSH from the phone

FREE
• Live streaming chat with your Hermes gateway
• Honest connection and run status (reconnecting, stale, unavailable — not a silent lie)
• QR and local pairing with no Hermes Mobile cloud account
• Saved computer profiles (home Wi‑Fi, Tailscale, or USB when the cable is data-ready)

LEASH PRO (19.99/mo)
• Review and approve or deny blocked tool calls from your phone
• Edit standing allow/block gate rules synced from your gateway
• Persistent safety policies across sessions
• ThumbGate feedback for assistant output and command decisions

FOR AI CODING AGENT OPERATORS
• Cursor and Claude Code: phone control surface for your Hermes gateway
• ChatGPT and Codex: supervise a multi-provider runtime you operate
• Gemini CLI, Copilot, Windsurf, OpenClaw: keep the executing computer and route visible
• Replit Agent: keep the runtime on your machine instead of a cloud IDE only

BUILT FOR REAL CONNECTIONS
• Home Wi‑Fi for local access
• Tailscale (or your tunnel) for cellular and away-from-home
• Clear reconnecting / stale / unavailable states
• API keys remain on the computer that runs the gateway

NOT THE SAME AS OPENCLAW MOBILE OR “HERMES ON PHONE”
• OpenClaw and phone-native agents run AI on the handset or their cloud
• Hermes Mobile is Method A control for a desktop Hermes gateway you host
• Termux: runs Hermes on Android. We do not — your gateway stays on the computer
• Hermes Agent (Hen Works) and similar: AI on the phone. We remote-control the desk
• Hermes-Relay style apps: often cloud “agent in pocket.” We pair to your gateway (QR / LAN / Tailscale)

SECURITY
• Camera only for pairing QR codes
• Optional product analytics can be disabled in Settings
• No ads and no IDFA tracking

Requires a computer running a compatible Hermes gateway. Remote access needs your own Wi‑Fi, Tailscale, tunnel, or relay.
