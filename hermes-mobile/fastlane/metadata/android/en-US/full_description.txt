Control an AI coding agent from your phone on your own Mac, Linux, or Windows computer. Chat with the Hermes gateway, see honest run state, and use Leash Pro to approve or deny risky commands before they execute.

Hermes Mobile is an independent mobile control plane for a Hermes gateway you operate. It is built for developers whose workflows also involve Cursor, Claude Code, ChatGPT or OpenAI Codex, Gemini CLI, Replit Agent, GitHub Copilot, Windsurf, OpenClaw, and other coding agents.

Hermes Mobile does not sign in to or replace those products. It solves a different job: supervising the Hermes agent and tools running through your own gateway from an Android phone.

FREE
• Live streaming chat with your Hermes gateway
• Current connection and run status instead of a false green state
• QR and local network pairing with no Hermes Mobile cloud account
• Saved computer profiles for Mac, Linux, and Windows gateways

LEASH PRO ($19.99/mo)
• Review and approve or deny blocked tool calls from your phone
• View and edit standing allow or block gate rules synced from your gateway
• Manage persistent safety policies across sessions
• Capture ThumbGate feedback for assistant output and command decisions

FOR AI CODING AGENT OPERATORS
• Cursor and Claude Code users: add a separate phone control surface for your Hermes gateway
• ChatGPT and Codex users: supervise a user-operated, multi-provider runtime instead of an OpenAI-only session
• Gemini CLI, GitHub Copilot, Windsurf, and OpenClaw users: keep the executing computer and connection route visible
• Replit Agent users: operate your own machine and gateway instead of moving the runtime into a cloud IDE

BUILT FOR REAL CONNECTIONS
• Wi-Fi and LAN for local access
• Tailscale or a configured tunnel for remote access
• Clear reconnecting, stale, and unavailable states
• Your API keys remain on the computer that runs the gateway

SECURITY
• Camera access is used only to scan pairing QR codes
• Optional product analytics can be disabled in Settings
• No ads and no IDFA tracking

Requires a computer running a compatible Hermes gateway. Remote access requires your own network, Tailscale, tunnel, or relay setup.
