Control an AI coding agent from your phone on your own Mac, Linux, or Windows computer. Chat with the Hermes gateway, see honest run state, and use Leash Pro to approve or deny risky commands before they execute.

Hermes Mobile is an independent operator client for a Hermes gateway you run. It is designed for developers whose workflows also involve Cursor, Claude Code, ChatGPT or OpenAI Codex, Gemini CLI, Replit Agent, GitHub Copilot, Windsurf, OpenClaw, and other coding agents.

Hermes Mobile does not sign in to or replace those products. It supervises the Hermes agent and tools running through your own gateway.

FREE
• Stream chat with your Hermes gateway
• See current connection and run state
• Pair a Mac, Linux, or Windows gateway with QR or local discovery
• Keep multiple computer profiles without a Hermes Mobile cloud account

LEASH PRO ($19.99/mo)
• Approve or deny blocked tool calls from your phone
• View and edit standing allow or block gate rules
• Manage persistent safety policies across sessions
• Capture ThumbGate feedback for assistant output and command decisions

BUILT FOR AI CODING AGENT OPERATORS
• Cursor and Claude Code users get a separate control surface for the Hermes gateway
• ChatGPT and Codex users can supervise a user-operated, multi-provider runtime
• Gemini CLI, Copilot, Windsurf, and OpenClaw users keep the executing computer visible
• Replit Agent users can operate their own machine instead of moving the runtime into a cloud IDE

REAL CONNECTION STATE
• Wi-Fi and LAN for local access
• Tailscale or a configured tunnel for remote access
• Clear reconnecting, stale, and unavailable states
• API keys remain on the computer that runs the gateway

Camera access is used only for pairing QR codes. Optional analytics can be disabled in Settings. No ads and no IDFA tracking.

Requires a compatible Hermes gateway. Remote access requires your own network, Tailscale, tunnel, or relay setup.
