# Enterprise multi-agent SDLC platforms — reliability gap

**Positioning (public-safe):** Enterprise platforms (e.g. IBM Bob-class multi-agent orchestration across discovery → plan → code → test → deploy) sell **coordination, governance, and cost analytics**. They do **not** replace **OS-level and loop-level enforcement** when agents thrash a Mac, burn tokens without finishing, or fail silently mid-workflow on non-mainframe fleets.
