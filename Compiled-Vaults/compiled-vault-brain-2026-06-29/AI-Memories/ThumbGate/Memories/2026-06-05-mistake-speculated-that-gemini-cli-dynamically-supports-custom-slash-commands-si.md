# MISTAKE: Speculated that Gemini CLI dynamically supports custom slash commands similarly to Claude Code, without checking binary...

What went wrong: Speculated that Gemini CLI dynamically supports custom slash commands similarly to Claude Code, without checking binary support. This led to false claims about functionality.
How to avoid: Verify support in target CLI before claiming a custom slash command will be recognized. Clearly communicate that for Gemini CLI, the dashboard must be run via terminal commands or shell shortcuts (like thumbgate-dashboard) rather than UI slash commands.
