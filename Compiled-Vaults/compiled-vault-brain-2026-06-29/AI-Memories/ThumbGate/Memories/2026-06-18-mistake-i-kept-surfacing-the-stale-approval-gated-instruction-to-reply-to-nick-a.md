# MISTAKE: I kept surfacing the stale approval-gated instruction to reply to Nick after the user reported they already sent the...

What went wrong: I kept surfacing the stale approval-gated instruction to reply to Nick after the user reported they already sent the reply.
How to avoid: When the user reports an external action is already done, stop prompting for the same approval. Verify the sent evidence if possible, then update the lead state and next-action card to await the prospect's response/materials.
