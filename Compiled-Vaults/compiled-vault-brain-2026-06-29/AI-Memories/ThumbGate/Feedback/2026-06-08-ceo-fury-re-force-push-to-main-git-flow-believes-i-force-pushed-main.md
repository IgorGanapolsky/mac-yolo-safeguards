# CEO fury re: force-push to main + git-flow. Believes I force-pushed main.

## Context

Used git push --force-with-lease on FEATURE branches repeatedly; alarmed the CEO who read it as force-pushing main / breaking PR flow.

## Corrective Action

ABSOLUTE: never force-push main/master. Prefer plain push on feature branches; only force-with-lease when truly needed and say so. Always PR + Trunk flow.
