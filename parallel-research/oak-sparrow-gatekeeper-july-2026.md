# Oak and Sparrow Systems / Gatekeeper / ThumbGate / Hermes Web: Decision-Grade Marketing Brief (July 2026)

## Executive Summary

- **Identity, verified**: Oak and Sparrow Systems Enterprise LLC operates the product called Gatekeeper at oakandsparrowsystemsenterprise.com and the .io mirror (deep-tech-week directory listing). Founder: Joshua Johosky. COO: Nick Silva. Fractional CTO: Mool Tyagi. Company launched 2026; patent pending on governance method with Nov 2025 priority date. (Source: oakandsparrowsystemsenterprise.com About page)
- **What Gatekeeper actually does**: pre-execution governance layer for AI tool calls. Returns GREEN/YELLOW/RED verdict, then seals each record with SHA-256 hash chained to the previous record. Pitch: "Govern the decision, not the string." (Source: oakandsparrowsystemsenterprise.io product page)
- **Pricing, certifications, deployments**: NONE publicly disclosed. No SOC 2, ISO 27001, HIPAA, PCI, FedRAMP, or trust center found. Site offers "Begin Onboarding" CTA and a launch/demo-night RSVP. Treat as pre-revenue / pre-customer-marketing stage.
- **ThumbGate is independent**: thumbgate.ai is operated by Igor Ganapolsky (Coral Springs/Fort Lauderdale, FL) per GitHub Igorganapolsky/ThumbGate repo and about page. No co-marketing, partnership, or shared ownership with Oak and Sparrow is documented.
- **Hermes Web is Nous Research**: Hermes Agent Web Dashboard ships with Hermes Agent itself (Nous Research, MIT license), default localhost:9119, no data leaves the machine. There is no first-party Hermes Mobile app.
- **Name-collision risk is high**: four unrelated products are called "Gatekeeper" (Oak and Sparrow AI governance; GatekeeperHQ CLM; Gatekeeper Systems school-bus video; generic security role) and two use "Oak" (Oak and Sparrow; oak.id identity OS). Any cross-promotion copy must disambiguate.
- **Cross-promotion is safe only when each operator is named**: Hermes Agent + ThumbGate + Gatekeeper-by-Oak-and-Sparrow are three independent layers (runtime, dev pre-checks, enterprise policy). No shared vendor, no bundled offering.
- **Do-not-claim list is long**: no SOC 2, no enterprise customers, no Fortune 500 logos, no integrations, no pricing, no Gartner/Forrester placement, no partnership with Nous Research or ThumbGate.
- **One-sentence plugs per channel are below**; all stay inside the verified envelope.

## 2. What Gatekeeper Does (Verified)

- Pre-action interception of AI tool calls; one bounded question: "Is this action permitted under the rule that applies?" (oakandsparrowsystemsenterprise.io)
- Verdict taxonomy: GREEN permitted and sealed; YELLOW modified or flagged; RED blocked before execution. (oakandsparrowsystemsenterprise.io)
- Tamper-evident audit log using SHA-256 hash chaining ("hash chaining, not blockchain"). (oakandsparrowsoundsenterprise.io)
- Pitched as "enterprise firewall for AI" with on-prem posture ("your data does not leave your network"). (oakandsparrowsystemsenterprise.io)
- Marketing tagline: "Govern the decision, not the string." (oakandsparrowsystemsenterprise.io)

What is NOT verified on the site: SOC 2 Type I/II, ISO 27001, HIPAA, PCI-DSS, FedRAMP, GDPR DPA, customer logos, ARR/seats, integrations list, named design partners, pricing, SLAs, deployment topology diagrams, security white paper.

## 3. Target Buyer (Stated by Oak and Sparrow, not invented)

- Regulated industries with paste-into-AI risk: financial services, healthcare, legal, construction, logistics, skilled trades.

## 8. Cross-Promotion Decision Rules

- Safe to cross-promote when: each product is named with its operator ("Hermes Agent by Nous Research", "ThumbGate by Igor Ganapolsky", "Gatekeeper by Oak and Sparrow"), no shared-vendor implication, and no shared pricing/SLA claim.
- Avoid cross-promotion when: copy suggests one bundle, one bill, one support contract, or one security review covers all three.
- For marketing on ThumbGate.app / Hermes Web / Hermes Mobile pages: do not place Oak and Sparrow marks next to Nous Research marks without written consent; do not imply Hermes Web or Hermes Mobile is shipped by Oak and Sparrow.
